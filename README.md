# luci-app-logcenter

OpenWrt Timeline Log Analyzer.

## Features

- Timeline View
- PPPoE Reconnect Analysis
- WAN Event Tracking
- SSH Log Classification
- DHCP Classification
- Firewall Event Tracking
- Realtime Log Stream

## Install

```bash
apk add luci-app-logcenter.apk
````

or

```bash
opkg install luci-app-logcenter.ipk
```

## Build

GitHub Actions supported.
